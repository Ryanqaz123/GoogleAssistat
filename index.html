FAQ

Welcome to Nimble!

What are its features?
Nimble is a site that allows you to input a Google Meet code or any link and have the page opened at a specific time. When it is time, Nimble will notify you with a gentle ring.

How do I use it?
Nimble is designed to be simple to learn and easy to use. Enter a Google meet code or a link to a site, give it a name, and enter a time you want the website to be opened. Then, press the plus button, and it should be added to the list of meets. The entries are automatically saved, so you do not have to input them again.

How do I enable pop-ups?
The easiest way is to press the button with the up-arrow icon. The website will try to open new windows and inform you of the result. If pop-ups are blocked, the web browser will prompt you to enable them.

How can I contact you?
