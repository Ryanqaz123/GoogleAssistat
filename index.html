Welcome to Nimble!

What are the features of Nimble?
Nimble is a site that allows you to input a Google Meet code or any link and select a time. Nimble will automatically open the site at the specified time and notify you with a gentle ring.

How do I use Nimble?
Nimble is designed to be simple to learn and easy to use. All you have to do it enter the Google meet code or a link to a site, give it a name, and enter a time you want the website to be opened. Then, press the plus button, and it should be added to the list of meets. The entries are automatically saved, so you don't have to input them again.

How do I allow pop-ups from Nimble?
The easiest way is to input a random name, link or code and set the time to 1 min in the future. If pop-ups are blocked, the web browser will prompt you to allow pop-ups from Nimble. All you need to do is allow pop-ups and you're good to go!
